# Focus Shield Extension Helper

This Chrome extension enhances Focus Shield's website blocking capabilities by integrating with BlockSite and providing stronger website blocking.

## Installation Instructions

### Step 1: Install the Focus Shield Helper Extension
1. Download this folder (extension-helper)
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (toggle in the top-right corner)
4. Click "Load unpacked" and select the downloaded folder
5. The Focus Shield Helper extension should now appear in your extensions list

### Step 2: Install BlockSite
1. Install BlockSite from the Chrome Web Store: [BlockSite Extension](https://chromewebstore.google.com/detail/blocksite-block-websites/eiimnmioipafcokbfikbljfdeojpcgbh)
2. Complete the BlockSite setup process

### Step 3: Verify the Integration
1. Open Focus Shield in your browser
2. You should see a confirmation message that BlockSite has been detected
3. When you start a focus session, websites will now be blocked by both Focus Shield and BlockSite for stronger protection

## Features
- Synchronizes blocked websites between Focus Shield and BlockSite
- Provides stronger blocking that prevents easy circumvention
- Automatically detects when BlockSite is installed
- Works even when you close and reopen your browser

## Troubleshooting
- If you don't see the BlockSite detected message, try refreshing the Focus Shield page
- Make sure both extensions are enabled in Chrome's extension manager
- If websites aren't being blocked, check that you've given BlockSite the necessary permissions

## Privacy
This extension does not collect or transmit any personal data. All blocking information is stored locally on your device.
